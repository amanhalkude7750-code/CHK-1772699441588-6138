Resume Builder

About Resume Builder

Resume Builder is a lightweight and beginner-friendly web application designed to help users create professional resumes with ease. With a minimal UI and intuitive design, users can input their personal, educational, and professional information and instantly preview it in a formatted resume template.

What You Can Do

  • Enter personal, educational, and job-related details
  • Choose from multiple resume templates (coming soon)
  • Live preview your resume on the right side
  • Download your resume as a PDF (optional add-on)

Features

  • Real-time resume preview
  • Fully responsive layout
  • HTML, CSS, and Vanilla JavaScript (no frameworks)
  • Clean and modern UI

Technologies Used

  • HTML5 — for page structure and forms
  • CSS3 — for layout and styling
  • JavaScript — for interactivity and live preview
